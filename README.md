# Reinforcement Learning AI Agent for Settlers of Catan 

## Project Overview  
This project focuses on developing an **AI agent** capable of autonomously playing **Settlers of Catan** using various **Reinforcement Learning (RL)** methods. The agent will learn optimal strategies through self-play and simulated games, leveraging reinforcement learning techniques.

## Tech Stack & Tools  
- **Python 3.12+** 
- **OpenAI Gym** (for custom game environment)

## Installation & Setup  

### 1️⃣ Clone the repository  
```bash
git clone https://github.com/j-coop/catan-rl.git
cd catan-rl
```

### 2️⃣ Create a virtual environment  
```bash
python -m venv venv
source venv/bin/activate  # Mac/Linux
venv\Scripts\activate  # Windows
```

### 3️⃣ Install dependencies  
```bash
pip install -r requirements.txt
```
